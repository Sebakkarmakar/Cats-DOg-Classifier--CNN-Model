# 🐱🐶 Cat vs Dog Image Classification (CNN - PyTorch)

## 📌 Project Overview
This project is a deep learning model built using a Convolutional Neural Network (CNN) to classify images of cats and dogs.

The model was trained on a custom dataset and improved using regularization and data augmentation techniques.

---

## 🚀 Features
- Built CNN model from scratch using PyTorch
- Image classification (Cat vs Dog)
- Data preprocessing and transformations
- Improved accuracy using Dropout and Data Augmentation
- Model evaluation and prediction

---

## 🧠 Model Architecture
- Conv2D → ReLU → MaxPooling
- Conv2D → ReLU → MaxPooling
- Flatten
- Fully Connected Layers
- Dropout for regularization

---

## 📊 Results
- Initial Accuracy: **70%**
- Final Accuracy: **73%** (after improvements)
- Better and more stable predictions after tuning

---

## 🛠️ Tech Stack
- Python
- PyTorch
